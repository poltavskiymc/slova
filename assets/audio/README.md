# Звуковые файлы для игры "Слова на лбу"

## Требуемые звуки:

### `beep.mp3` (переименован из old-ships-bell-ding.mp3)
- Звук корабельного колокола (для окончания игры)
- Длительность воспроизведения: 2.5 секунды (первые 2.5 сек из оригинала)
- Формат: MP3, стерео, 44.1kHz

## Рекомендации:

1. **Формат:** MP3 (лучшая совместимость)
2. **Каналы:** Моно (экономит место)
3. **Частота дискретизации:** 44.1kHz
4. **Битрейт:** 128-192 kbps
5. **Громкость:** Нормализованная (-6dB до 0dB)

## Как добавить звук:

1. Положите файл `beep.mp3` в папку `assets/audio/`
2. Убедитесь, что в `pubspec.yaml` указаны assets:
   ```yaml
   flutter:
     assets:
       - assets/audio/
   ```
3. Запустите `flutter pub get`

## Примеры звуков:

Можно использовать:
- Звук будильника
- Звук гонга
- Звук колокольчика
- Любой короткий оповещающий звук
